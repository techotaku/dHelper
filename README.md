微信：豆瓣查
=======

豆瓣查：豆瓣影音书微信速查入口。  
（拍了张手持身份证的照片……然后发现哦艹原来个人只能申请订阅号。）  

## 使用说明

关注微信账号“doubancha”（或前往[此页面](http://dbcha.techotaku.net/)扫描二维码），发送指令进行查询。

指令说明：  

	类型 关键字

可用类型包括：电影、f；音乐、m；图书、b。  
如"电影 一代宗师"，"b 九州 天光云影"。
